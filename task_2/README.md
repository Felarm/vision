Скрипт запускается командой python3 script.py *путь до папки*
Функция upload_image принимает на вход путь до папки из командной строки и URL, который задаётся в самом скрипте.
Затем она проверяет валидность пути до папки и создаёт список строк с именами файлов внутри.
Затем для каждого элемента списка в цикле создаётся абсолютный путь до файла и он проверяется изображение ли это с 
помощью библиотеки imghdr, функция из которой возвращает расширение файла изображения или None, если это не изображение
Если файл оказывается изображением, то оно кодируется в base64 (эту кодировку принято использовать для передачи изображений) 
и создается объект Request, которому затем добавляется заголовок  'Content-Type:image/*формат изображения*' и 
отправляется POST запрос на сервер.

Для проверки работоспособности можно использовать скрипт test_serv.py, написанный с использованием Flask'a
